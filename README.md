# BancoDoMicro
## Banco monolítico transformado em micro serviço

Esse é o front-end da aplicação que foi desmembrada de uma arquitetura monolítica para uma arquitetura de micro serviços MVC, com implantação na nuvem AWS. 

Caso queira saber mais sobre o projeto acesse o site:
http://ec2-3-20-224-158.us-east-2.compute.amazonaws.com/
